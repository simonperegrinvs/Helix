# Glossary

- **Vault-first**: canonical project knowledge lives in Obsidian folder structure.
- **Canonical content**: durable markdown/files in project vault.
- **Derived index**: rebuildable retrieval index, not source of truth.
- **Grounded answer**: response linked to project evidence with citations.
- **Hypothesis**: claim lacking sufficient support, explicitly labeled.
- **MCP secondary ingress**: automation interface over existing services.
